# Paralang

Paralang is a local bilingual HTML comparison tool for reviewing English and French versions of web pages side by side.

## Features

- Side-by-side EN/FR page view
- Code view
- Manual sync offset controls
- Page/code comparison workflow
- Local Flask app

## Work in progress

- Local translation MiniLM implementation (sentence-transformers)
    - Match EN/FR equivalent elements
    - Improve scroll-syncing
    - Identify questionable translations
    - Find and highlight missing/merged elements
- And much more...
